Paso 1: Descomprimir los recursos, se recomienda el directorio /Documentos

Paso 2: Descargar e instalar Anaconda (Python + Jupyter)

Comenzamos yendo a la web siguiente https://www.continuum.io/downloads y descargamos el instalador, es un instalador normal en .pkg:

A continuación abrimos el instalador y seguimos los pasos, hasta el final, tal como muestra esta animación:

Cuando finalice (tarda un buen rato) ya tendremos instalado Python y Jupyter.  

Paso 3: Abrir una terminal y comprobar todo

Bien, para abrir una terminal rápidamente presionamos la combinación ⌘ + Espacio y escribimos terminal. 

Una vez abierta escribimos python -V, si muestra un mensaje parecido al siguiente estamos preparados para continuar:

A continuación navegamos a nuestro directorio CursoPython (utilizando el comando cd), y una vez dentro el comando para abrir el notebook es jupyter notebook. Recordádlo bien porque tendréis que ejecutarlo siempre antes de comenzar a trabajar:

Automáticamente al ejecutarlo se abrirá Jupyter Notebook en vuestro navegador:

Con ésto ya lo tenemos listo. Recordad no cerrar la terminal, porque si lo hacéis Jupyter dejará de funcionar.

Paso 4 (Optativo): Crear un acceso directo a Jupyter Notebook para abrirlo haciendo doble clic.

Si en lugar de andar escribiendo "jupyter notebook" en la terminal, prefieres crear un acceso directo, entonces sólo debes crear un fichero con tu editor de texto llamado notebook.command en el escritorio, y ahora suponiendo que tienes el directorio CursoPython en Documentos, añade el siguiendo código y guárdalo:

#!/bin/bash
cd ~/Documents/CursoPython;
jupyter notebook;
exit;

Ahora tienes que darle el permiso de ejecución al fichero para poder hacerle doble clic, abre una terminal y escribe (suponiendo que tienes el fichero en el escritorio):

chmod +x /Desktop/notebook.command

Con esto hecho ya podrás poner en marcha fácilmente Jupyter Notebook haciendo doble clic en tu acceso directo, pero asegúrate de no tenerlo ya en marcha ;-)

Paso 5: Descargar e instalar el editor SublimeText

Vamos a la web del editor https://www.sublimetext.com/3 y descargamos el instalador en formato .dmg. Luego hacemos doble clic, seguimos los pasos y lo instalamos.

Una vez instalado podemos hacer la combinación ⌘ + Espacio y escribir SublimeText para abrirlo rápidamente.

Por último recordad la combinación ⌘ + B (Build), la utilizaremos más adelante para ejecutar código Python directamente.
